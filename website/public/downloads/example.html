Accessibility Widget v1 — Example Page

Welcome! This page demonstrates the Accessibility Widget v1 in action.

Look for the accessibility widget button in the top-right corner. Click it to adjust:

  • Contrast modes (Default, High contrast, Dark, Light)
  • Text size (100% to 160%)
  • Text spacing (Normal, Comfortable, Max)
  • Readable font toggle
  • Reduce motion toggle

All settings persist across page reloads!

Widget Features

  • Contrast Modes: Choose from Default, High contrast, Dark, or Light themes
  • Text Scaling: Adjust text size from 100% to 160%
  • Text Spacing: Normal, Comfortable, or Max spacing options
  • Readable Font: Toggle system-friendly sans-serif font
  • Reduce Motion: Disable animations and transitions
  • Presets: Quick access to Low vision, Dyslexia-friendly, and Reduced motion presets
  • Reset: Return all settings to defaults

Note: This example uses CDN paths by default. The widget should work immediately when you open this file.

To use local files:

  1. Download a11y-widget.css and a11y-widget.js from the repository
  2. Place them in the same folder as this HTML file
  3. Uncomment the local file paths in the <head> section
  4. Comment out the CDN paths

For production: Update the paths to point to your own CDN:

<link rel="stylesheet" href="https://cdn.YOURDOMAIN.com/a11y-widget/v1/a11y-widget.css" />
<script src="https://cdn.YOURDOMAIN.com/a11y-widget/v1/a11y-widget.js" defer></script>

Testing the Widget

Try these actions to test the widget:

  1. Click the widget button in the top-right corner
  2. Change the contrast mode and observe the color changes
  3. Adjust the text size slider and see how content scales
  4. Try different spacing presets (Normal, Comfortable, Max)
  5. Enable "Readable font" to see the font change
  6. Try the preset buttons (Low vision, Dyslexia-friendly, Reduced motion)
  7. Reload the page — your settings should persist!
  8. Click "Reset settings" to return to defaults
